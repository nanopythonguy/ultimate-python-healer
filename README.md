# ultimate-python-healer
This program fully written on python is a gathering of Windows (and maybe mine) tools used for detecting and helping against some viruses or lags.

To use the program run it with administrator privileges.

Avaliable functions(for now):
    sfc /scannow
    clear recycle bin
    clear temp folder

In development:
    error handler
    more functions
    progress bars 
    and more!
